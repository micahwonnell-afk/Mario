# Mario Bros Game Instructions

## Objective
The objective of the game is to navigate Mario through various levels, overcoming obstacles and defeating enemies to rescue Princess Peach.

## Controls
- **Arrow keys**: Move Mario left and right.
- **Space bar**: Jump.
- **Shift**: Run faster.

## Gameplay
1. Start the game by pressing **Enter**.
2. Use the arrow keys to move Mario through the levels.
3. Jump on enemies to defeat them.
4. Collect coins for points.
5. Reach the end of the level to advance to the next one.

## Tips
- Try to collect power-ups to gain special abilities.
- Avoid falling into pits and touching harmful enemies.

Good luck and have fun!
